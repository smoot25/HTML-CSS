속성선택자  
  1. 속성선택자 : [속성명]
    • * : 값의 포함 여부
    • ^ : 값의 시작 여부
    • $ : 값의 끝여부
  2. 하위 요소 선택자 : 공백
  3. 자식 요소 선택자 : >
  4. 인접 형제 선택자 : +
  5. 형제 요소 선택자 : ~


   EXAMPLE   

마루는 황준호의 집에서 키우고 있는 애완동물 강아지이며, 올해로 5세 이다.

가장 좋아하는 음식은 고구마이며, 닭고기에는 알레르기가 있다.

마루는 겁이 굉장히 많아서 산책 나가는 것을 그렇게 좋아하지는 않는다.

그래도 집에 옥상이 있어서 틈만 나면 마루를 옥상에서 뛰어놀도록 한다.



   의사클래스   

의사클래스 : 사용자의 행동으로 변경되는 css

사용방법 = 선택자 : 의사클래스

의사클래스의 종류

  • hover : 마우스 커서 올리기
  • active : 마우스로 클릭
  • focus : 입력창 깜빡거리는 효과
  • disabled : 입력요소에 입력 불가
  • nth-child() : 형제중에 몇번째 형제를 선택한다.

   의사요소   

의사요소 : 요소의 특정부분만을 선택가능

사용방법 = 선택자 :: 의사요소

의사요소의 종류

  • after : 요소 앞에 새콘텐츠 추가후 선택
  • before : 요소 뒤에 새콘텐츠 추가후 선택
  • first-line
  • marker
  • placeholder
